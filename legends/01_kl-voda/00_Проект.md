## RedRamka

### Проект: Платформа автоматизации доставки питьевой воды ключевая вода

#### Описание системы

Серверная часть мобильного приложения для автоматизации работы сервиса доставки питьевой воды и сопутствующих товаров (B2C e-commerce).

Система используется экспедиторами и операторами компании для управления заказами, учёта складских остатков, проведения предрейсовых процедур (техосмотр, медосмотр), обработки доставок и сдачи выручки. Backend обеспечивает обработку заказов, интеграцию с учетной системой предприятия и поддержку бизнес-логики мобильного приложения.

#### Архитектура

Ключевые компоненты системы:

* Backend API сервис на PHP (Laravel) для мобильного приложения
* PostgreSQL в качестве основной транзакционной базы данных
* Redis для кэширования данных и ускорения часто используемых операций
* RabbitMQ для асинхронной обработки задач и ретраев интеграций
* Интеграция с ERP системой предприятия (1С) через REST API
* Интеграция с Yandex Suggestions API для нормализации адресов
* Мониторинг метрик и состояния системы через Prometheus и Grafana

Взаимодействие построено по принципу REST API между мобильным клиентом и backend сервисом. Интеграции с внешними системами и обработка ошибок вынесены в асинхронные очереди RabbitMQ для повышения отказоустойчивости.

#### Нагрузка и данные

* До 1600 RPS пиковых запросов API
* Более 2500 активных пользователей системы
* Высокая конкурентная нагрузка на операции оформления и обработки заказов
* Асинхронные очереди RabbitMQ для обработки задач интеграции и повторных попыток
* Основные доменные сущности: заказы, клиенты, складские остатки, акции и тарифы

#### Моя зона ответственности

Разработка и развитие серверной бизнес-логики системы.

Отвечал за:

* интеграции с внешними сервисами и ERP системой
* архитектуру обработки заказов и складских операций
* оптимизацию производительности API
* внедрение очередей для асинхронной обработки
* обеспечение корректности операций при конкурентной нагрузке

Участвовал в CI/CD процессах, покрытии системы тестами, рефакторинге и развитии архитектуры backend-сервисов.


#### Технологии

PHP, Laravel, PostgreSQL, Redis, RabbitMQ, Docker, PHPUnit, Swagger, Prometheus, Grafana.

#### Результаты

* Повышена устойчивость интеграции с ERP системой за счёт асинхронной обработки и механизма повторных попыток.
* Обеспечена корректность складских операций при высокой конкурентной нагрузке.
* Ускорено оформление заказов благодаря автоматическому определению адресных данных.
* Снижено время обнаружения инцидентов за счёт внедрения мониторинга API.
